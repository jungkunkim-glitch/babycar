# 🚗 마이 드림카 키우기 - CBTI 게임

해커톤용 자동차 육성 게임입니다. 고객의 성향을 파악하여 맞춤 차량과 금융 상품을 추천합니다.

## 🎮 게임 특징

- **플레이 타임**: 5분 내외
- **CBTI 시스템**: 32가지 유형 분석 (5가지 성향 축)
- **포인트 시스템**: 게임 플레이로 실제 금리 할인 혜택 획득
- **모바일 최적화**: 반응형 디자인

## 🚀 실행 방법

### 방법 1: 직접 브라우저에서 열기 (가장 쉬움!)

1. `index.html` 파일을 더블클릭
2. 또는 브라우저 주소창에 파일 경로 입력

### 방법 2: 간단한 로컬 서버 실행

Python이 설치되어 있다면:

```bash
# Python 3
python3 -m http.server 8000

# 또는 Python 2
python -m SimpleHTTPServer 8000
```

그 다음 브라우저에서 http://localhost:8000 접속

### 방법 3: VS Code Live Server

1. VS Code에서 프로젝트 열기
2. `index.html` 우클릭
3. "Open with Live Server" 선택

## 🎯 게임 흐름

1. **인트로** - 시작 화면
2. **알 선택** - 3가지 알 중 선택 (활동성향 파악)
3. **성장 미션** - 5개 질문으로 성향 파악
   - 밥주기 (차량이용 + 소비성향)
   - 산책 스타일 (가족성향 + 활동성향)
   - 목적지 선택 (활동성향 + 소비성향)
   - 사고 대응 (금융선호 + 소비성향)
   - 업그레이드 (소비성향)
4. **결과** - CBTI 결과 + 차량 추천 + 금융 상품

## 📊 성향 분석 체계

### 5가지 성향 축
1. **차량이용성향**: EV ↔ 내연기관
2. **소비성향**: 고급지향 ↔ 실속지향
3. **활동성향**: 활동적 ↔ 루틴
4. **가족성향**: 가족중심 ↔ 개인중심
5. **금융선호**: 소유(할부) ↔ 임대(리스)

## 💰 포인트 시스템

- 시작: 100P
- 알 선택: +50P
- 각 질문 답변: +50P
- 총 획득 가능: 400P

### 포인트별 할인 혜택
- 300~399P: 금리 -0.2%
- 400~499P: 금리 -0.3%
- 500~599P: 금리 -0.5%
- 600P 이상: 금리 -0.7%

## 🎨 커스터마이징

### 색상 변경
`css/style.css` 파일에서 그라데이션 색상을 변경할 수 있습니다.

```css
/* 예시: 메인 배경 색상 */
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### 질문 추가/수정
`js/data.js` 파일의 `QUESTIONS` 배열을 수정하세요.

```javascript
{
    id: 'new-question',
    question: '새로운 질문',
    options: [
        {
            text: '옵션 1',
            scores: { vehicle: 2, spending: 1 }
        }
    ]
}
```

### 차량 추가
`js/data.js` 파일의 `VEHICLES` 객체에 추가하세요.

```javascript
'NEW_VEHICLE': { brand: '브랜드', model: '모델명' }
```

## 🐛 문제 해결

### 화면이 안 나와요
- 브라우저 개발자 도구 (F12) → Console 탭에서 에러 확인
- 파일 경로가 올바른지 확인

### 버튼이 작동하지 않아요
- JavaScript가 활성화되어 있는지 확인
- 모든 `.js` 파일이 올바르게 로드되었는지 확인

### 모바일에서 레이아웃이 깨져요
- 뷰포트 메타 태그가 있는지 확인
- CSS 미디어 쿼리 확인
