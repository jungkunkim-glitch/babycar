# 🚗 마이 드림카 키우기 - CBTI 게임 시나리오

## 게임 개요
- **플레이 타임**: 5분 내외
- **목적**: 고객 성향 파악(CBTI) + 포인트 획득 + 금융상품 매칭
- **결과**: 32가지 유형 (16개 차종 × 2개 금융옵션)

---

## 📊 CBTI 성향 파악 체계

### 5가지 성향 축
1. **차량이용성향**: EV 선호 ↔ 내연기관 선호
2. **소비성향**: 고급지향 ↔ 실속지향
3. **활동성향**: 활동적 라이프 ↔ 루틴 라이프
4. **가족/가구성향**: 가족중심형 ↔ 개인중심형
5. **금융선호**: 소유(할부) ↔ 임대(리스/렌트)

---

## 🎮 Stage 1: 인트로 & 알 선택 (45초)

### Scene 1-1: 오프닝 (10초)
**화면 구성**:
- 배경: 은은한 그라데이션 배경 (하늘 → 도로)
- 중앙: "당신만의 드림카를 키워보세요!" 타이포 애니메이션
- 하단: "시작하기" 버튼 (펄스 효과)

**인터랙션**:
- "시작하기" 버튼 터치 → Scene 1-2로 전환
- 배경음악: 경쾌한 신스팝 인트로

**포인트 획득**: 
- 게임 시작 웰컴 보너스 +100P

---

### Scene 1-2: 알 선택 (35초)
**화면 구성**:
- 상단: "당신의 꿈의 자동차가 잠들어있어요. 알을 선택해주세요!"
- 중앙: 3개의 알이 회전하며 등장
- 우상단: 포인트 카운터 (100P)
- 하단: 선택 확인 버튼

**알 3가지 옵션** (활동성향 파악):

1. **🔥 파이어 에그** (빨간색, 불꽃 이펙트)
   - 설명: "열정적이고 역동적인 당신에게"
   - 선택 시: 활동성향 +3점 (활동적)
   - 등장 애니메이션: 튀어오르며 회전

2. **🌊 아쿠아 에그** (파란색, 물결 이펙트)
   - 설명: "차분하고 안정적인 당신에게"
   - 선택 시: 활동성향 -3점 (루틴)
   - 등장 애니메이션: 부드럽게 부유

3. **⚡ 썬더 에그** (노란색, 번개 이펙트)
   - 설명: "밸런스 있는 라이프스타일을 원하는 당신에게"
   - 선택 시: 활동성향 0점 (중립)
   - 등장 애니메이션: 빠르게 깜빡임

**인터랙션**:
- 각 알을 터치하면 확대되며 설명 표시
- 선택 후 "이 알로 시작하기" 버튼 활성화
- 확인 버튼 터치 → 알 부화 애니메이션 (3초)
- 알에서 빛이 터지며 아기차 등장!

**포인트 획득**:
- 알 선택 완료 보너스 +50P (총 150P)

**부화 연출**:
- 알이 흔들리며 금이 감
- 빛이 터지며 귀여운 "삐빅~" 사운드
- 작은 자동차(아기차)가 통통 튀며 등장

## 🌱 Stage 2: 성장 미션 (90초)

### Scene 2-1: 아기차 소개 (15초)
**화면 구성**:
- 중앙: 귀여운 아기차 캐릭터 (큰 헤드라이트 눈, 작은 차체)
- 상단: "당신의 아기차가 태어났어요! 🎉"
- 말풍선: "안녕! 나를 키워줄래?"
- 하단: "성장 시작하기" 버튼

**게임 UI 안내**:
- 좌상단: 성장 게이지 (0% → 100%)
- 우상단: 포인트 카운터
- 하단: 액션 버튼 영역

---

### Scene 2-2: 밥주기 미션 (30초)
**화면 구성**:
- 중앙: 아기차가 배고파하는 애니메이션 (배터리/연료 게이지 깜빡임)
- 말풍선: "배고파요... 🥺"
- 하단: 2가지 선택지

**선택지** (차량이용성향 + 소비성향 파악):

**Option A: 전기 충전 ⚡**
- 버튼: "전기 충전하기" (녹색 버튼, 플러그 아이콘)
- 하위 선택:
  - "급속 충전 (프리미엄)" → EV +3, 고급지향 +2
  - "완속 충전 (경제적)" → EV +3, 실속지향 +2
  
**Option B: 연료 주입 ⛽**
- 버튼: "연료 주입하기" (빨간색 버튼, 주유기 아이콘)
- 하위 선택:
  - "프리미엄 휘발유" → 내연기관 +3, 고급지향 +2
  - "일반 휘발유" → 내연기관 +3, 실속지향 +2

**인터랙션**:
- 선택 시 충전/주입 애니메이션 (2초)
- 아기차가 "냠냠" 소리와 함께 만족하는 표정
- 성장 게이지 +20% 증가

**포인트 획득**:
- 밥주기 완료 +30P

---

### Scene 2-3: 산책하기 미션 (45초)
**화면 구성**:
- 배경: 도로 스크롤 애니메이션
- 중앙: 아기차가 운전하는 모습
- 말풍선: "밖에 나가고 싶어요!"

**선택지** (가족/가구성향 파악):

**첫 번째 질문: 산책 스타일**
- "혼자 드라이브 🏍️" → 개인중심 +3
- "가족과 함께 드라이브 👨‍👩‍👧‍👦" → 가족중심 +3

**두 번째 질문: 산책 목적지**
선택에 따라 추가 성향 파악:

**만약 "혼자 드라이브" 선택 시**:
- "짜릿한 산길/해안도로" → 활동적 +2
- "익숙한 동네 한 바퀴" → 루틴 +2

**만약 "가족과 함께" 선택 시**:
- "놀이공원/테마파크" → 활동적 +2, 고급지향 +1
- "가까운 마트/공원" → 루틴 +2, 실속지향 +1

**인터랙션**:
- 선택 후 3초간 주행 애니메이션
- 배경이 스크롤되며 선택한 목적지 풍경 표시
- 아기차가 즐거워하는 애니메이션 (흔들흔들)
- 성장 게이지 +30% 증가

**포인트 획득**:
- 산책 완료 +50P

## 🚨 Stage 3: 이벤트 처리 (60초)

### Scene 3-1: 성장 중 랜덤 이벤트
**화면 구성**:
- 경고음과 함께 화면 흔들림 효과
- 중앙: 이벤트 발생 팝업

**이벤트 시나리오**:

#### 이벤트 1: 가벼운 접촉 사고 (금융선호 파악)
**상황 설명**:
- "앗! 주차장에서 살짝 긁혔어요 😱"
- 수리비 예상: 50만원
- 아기차가 슬픈 표정

**선택지**:
1. **"바로 완벽하게 수리하기"** 
   - 소유(할부) +3, 고급지향 +1
   - 비용: -20P (수리비)
   - 결과: 아기차가 새것처럼 반짝반짝

2. **"저렴한 곳에서 간단히 수리"**
   - 소유(할부) +2, 실속지향 +2
   - 비용: -10P
   - 결과: 아기차가 만족하지만 약간 흠집 남음

3. **"일단 더 타보고 나중에 수리"**
   - 임대(리스/렌트) +3
   - 비용: 0P
   - 결과: 아기차가 약간 불안해하지만 계속 주행

**포인트 획득**:
- 이벤트 처리 보너스 +40P

---

#### 이벤트 2: 성장 선택의 순간 (소비성향 추가 파악)
**상황 설명**:
- "아기차가 청소년기로 성장했어요! 🚗"
- 성장 게이지 70% 도달
- 업그레이드 옵션 제시

**선택지**:
1. **"최고급 부품으로 튜닝하기"**
   - 고급지향 +3
   - 비용: -30P
   - 결과: 반짝이는 크롬 휠, LED 라이트 장착

2. **"실용적인 옵션 추가하기"**
   - 실속지향 +3
   - 비용: -15P
   - 결과: 연비 향상, 편의 기능 추가

3. **"지금 모습 그대로 키우기"**
   - 중립
   - 비용: 0P
   - 결과: 순정미 유지

**인터랙션**:
- 선택 시 변신 애니메이션 (3초)
- 빛이 감싸며 차가 조금 더 성장
- 성장 게이지 +20% 증가

**포인트 획득**:
- 성장 단계 돌파 보너스 +60P

## 🎯 Stage 4: 미니게임 & 최종 성장 (90초)

### Scene 4-1: 부품 수집 게임 (40초)
**게임 방식**: 간단한 러닝 게임
**화면 구성**:
- 배경: 도로가 자동 스크롤
- 중앙: 내 차가 좌우 이동
- 도로에 아이템들이 등장

**수집 아이템** (차량이용성향 & 소비성향 추가 파악):

**좋은 아이템**:
- ⚡ **배터리 팩**: +10P, EV +1
- ⛽ **연료통**: +10P, 내연기관 +1
- 💎 **프리미엄 파츠**: +15P, 고급지향 +1
- 🔧 **실속 파츠**: +10P, 실속지향 +1
- ⭐ **보너스 스타**: +20P

**피해야 할 아이템**:
- 🚧 **공사 콘**: -5P
- 💥 **장애물**: -10P

**조작법**:
- 좌우 터치 또는 기울이기로 차량 이동
- 자동으로 전진

**인터랙션**:
- 30초간 플레이
- 획득한 아이템에 따라 포인트 누적
- 성장 게이지 +15% 증가

**포인트 획득**:
- 기본 완료 보너스 +40P
- 수집 아이템에 따라 추가 포인트

---

### Scene 4-2: 간단한 레이싱 챌린지 (50초)
**게임 방식**: 탭 타이밍 게임
**화면 구성**:
- 배경: 레이싱 트랙
- 중앙: 신호등 & 내 차
- 하단: 탭 타이밍 게이지

**게임 플레이**:
1. **준비 단계**:
   - "레이스에 참가하시겠어요?" 
   - 차량 선택지:
     - "고성능 스포츠카로" → 활동적 +2, 개인중심 +1
     - "안정적인 세단으로" → 루틴 +2, 가족중심 +1

2. **레이싱 단계**:
   - 신호등: 빨강 → 노랑 → 초록!
   - 초록불 때 정확히 탭 → Perfect! +50P
   - 약간 빠름/늦음 → Good! +30P
   - 너무 빠름/늦음 → Miss! +10P

3. **결과**:
   - Perfect: "1등! 완벽한 드라이빙!"
   - Good: "2등! 훌륭해요!"
   - Miss: "완주! 경험 획득!"

**인터랙션**:
- 3번의 탭 기회 제공
- 최고 점수 기록
- 성장 게이지 +15% → **100% 완성!**

**포인트 획득**:
- 레이싱 완료 보너스 +30P ~ 80P (성적에 따라)

---

- 아기차 → 성인차 변신 애니메이션
- 폭죽 효과

**최종 성장 연출**:
- 성장 게이지 100% 도달
- 차량이 완전체로 변신
- 축하 사운드 & 진동 피드백

각 성향은 -10 ~ +10 점 범위:

- +6 이상: EV 강선호
- +3~+5: EV 선호

- -5~-3: 내연기관 선호
- -6 이하: 내연기관 강선호

**2. 소비성향**
- +6 이상: 고급지향
- +3~+5: 준고급
- -2~+2: 중간
- -5~-3: 실속지향
- -6 이하: 강실속지향

**3. 활동성향**
- +6 이상: 매우 활동적 (스포츠카, SUV)
- +3~+5: 활동적
- -2~+2: 밸런스
- -5~-3: 루틴
- -6 이하: 안정 지향

**4. 가족/가구성향**

- +3~+5: 가족 고려
- -2~+2: 다목적
- -5~-3: 개인 중심
- -6 이하: 개인 최적화

**5. 금융선호**
- +6 이상: 소유 강선호 (장기할부)
- +3~+5: 소유 선호
- -2~+2: 유연
- -5~-3: 임대 선호 (리스)
